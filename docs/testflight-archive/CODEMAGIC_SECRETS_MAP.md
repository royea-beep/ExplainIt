# Codemagic Secrets Map — ExplainIt

## Required Secrets & Variables

| Secret / Variable | Where to create it | Where to paste in Codemagic | Required? | Format | Notes |
|-------------------|--------------------|-----------------------------|-----------|--------|-------|
| **Issuer ID** | App Store Connect → Users and Access → Integrations → API (shown at top) | Teams → Integrations → App Store Connect → "Issuer ID" field | Yes | UUID: `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx` | Same for all keys in your account |
| **Key ID** | App Store Connect → Integrations → API (shown next to key name) | Teams → Integrations → App Store Connect → "Key ID" field | Yes | 10-char alphanumeric: `ABC1234DEF` | Unique per key |
| **API Key (.p8)** | Downloaded once when key is created in App Store Connect | Teams → Integrations → App Store Connect → "API Key" file upload | Yes | `.p8` file (PEM-encoded private key) | Upload the file directly — do NOT paste contents |
| **APP_STORE_APP_ID** | App Store Connect → your app → General → Apple ID | App settings → Environment variables → Add variable | Yes | Numeric: `1234567890` | Not a secret — can be "not secure" |

## NOT Required (Codemagic handles automatically)

| Item | Why not needed |
|------|---------------|
| Apple Team ID | Resolved automatically from the API key |
| BUNDLE_IDENTIFIER | Already in `codemagic.yaml` (`com.explainit.app`) |
| Signing certificates | Auto-generated by Codemagic from API key |
| Provisioning profiles | Auto-generated by Codemagic from API key |
| DEVELOPMENT_TEAM in Xcode | Overridden by `xcode-project use-profiles` |
| APPLE_CERTIFICATE / P12 | Not needed — Codemagic uses App Store Connect API signing |

## How Codemagic Uses These

```
API Key (.p8) + Issuer ID + Key ID
    → Codemagic authenticates to App Store Connect API
    → Auto-creates signing certificate
    → Auto-creates provisioning profile for com.explainit.app
    → Signs the .ipa
    → Uploads to TestFlight

APP_STORE_APP_ID
    → Used by app-store-connect CLI to fetch latest build number
    → Increments build number for each upload
    → If missing: falls back to build number 1 (first build works fine)
```

## Step-by-Step: Adding Secrets to Codemagic

### 1. App Store Connect Integration (one-time, team-level)
1. Go to codemagic.io → **Teams** → your team → **Integrations**
2. Find "App Store Connect" → click **Connect**
3. Enter **Issuer ID** (UUID from App Store Connect)
4. Enter **Key ID** (10-char string from App Store Connect)
5. Click **Choose file** → upload the `.p8` file
6. Click **Save**

### 2. APP_STORE_APP_ID (per-app, app-level)
1. Go to codemagic.io → your app → **Environment variables**
2. Click **Add variable**
3. Variable name: `APP_STORE_APP_ID`
4. Variable value: the numeric Apple ID from App Store Connect
5. Group: leave default
6. Secure: No (it's not a secret)
7. Click **Add**
